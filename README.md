# sprint-9-project

Репозиторий для разработки проекта 9-го спринта по облачным технологиям.

## Описание проекта

В рамках проекта реализованы два микросервиса:

- **service_dds** — сервис, который читает данные из Kafka, загружает их в слой **DDS** хранилища и отправляет обработанные данные дальше в Kafka.
- **service_cdm** — сервис, который читает данные из Kafka и загружает агрегированные данные в слой **CDM**.

Проект использует:

- **Kafka** — для потоковой передачи данных;
- **PostgreSQL** — в качестве DWH;
- **Docker** — для контейнеризации сервисов;
- **Kubernetes** — для деплоя сервисов;
- **Yandex Cloud Container Registry** — для хранения Docker-образов.
